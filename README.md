# Car-Dataset-Visualization-personal-project-

## Overview
This project analyzes a CSV Car dataset to identify trends and support decision-making.

## Tools Used
- Power BI(Mostly)
- Excel

## Key Insights
- Visualize for easy understanding of the data
- VENENO ROADSTER was the most purchased car
- Ferari was the most bought from company
- The car's horsepower determines the car's price
